Physics-Based Particle Simulation with Terrain Collision

Python Nvidia Warp Physics Simulation

A physics-based particle system that simulates real-time particle motion and terrain interaction using numerical integration and collision response. The system supports configurable simulation parameters and exports animated results to USD for 3D visualization.

Features

  • Particle motion using numerical time integration
  • Terrain collision detection and response
  • Adjustable restitution and friction parameters
  • Configurable particle counts and simulation length

How It Works

Particles are simulated using discrete time integration. Collisions against terrain geometry are resolved using surface normals and velocity correction to produce realistic bounce and sliding behavior.

The system is designed for experimentation with physical parameters and scalable particle counts.

Core Technologies

  • Python
  • Nvidia Warp (physics kernel acceleration)

Running the Simulation

conda create -n particle-sim -c conda-forge python=3.14 numpy pip openusd -y
conda activate particle-sim
pip install warp-lang
python src/main.py --n 100 --frames 300 --integrator forward --out simulation.usd --dt 0.01 --fps 24 --restitution 0.5 --friction 0.05

Future Improvements
