Face Animation with Blend Shapes

Python Nvidia Warp GPU Computing Computer Graphics

A GPU-accelerated facial animation system that uses blend shapes to create smooth transitions between facial expressions. The system implements multiple interpolation methods and custom easing functions for natural-looking animations.

Features

  • GPU-accelerated blend shape deformation using Nvidia Warp
  • Interactive slider interface for real-time expression control
  • Linear interpolation for expression morphing
  • Catmull-Rom and Hermite spline interpolation for smoother transitions
  • Custom cubic and bounce easing functions (ease-in, ease-out, ease-in-out)

Core Technologies

  • Python
  • Nvidia Warp (GPU kernel acceleration)
  • Polyscope (3D visualization)
  • NumPy (numerical computations)

How It Works

The system starts with a base mesh and applies weighted deltas to deform the face into different expressions. Each vertex position is computed in parallel on the GPU.

Interpolation Methods

Linear Interpolation (LERP): Blends between two consecutive expressions by calculating a weighted average based on time, creating straight, constant-speed transitions between keyframes.

Catmull-Rom Spline: Uses four control points to create smooth, curved transitions that pass exactly through each expression keyframe, resulting in more natural-looking animations.

Hermite Spline: Uses two control points along with their tangent vectors to define the curve. This gives precise control over how the animation enters and exits each keyframe, allowing for customized easing at each transition point.

Easing Functions: Modifies the timing of transitions to create more dynamic animations. Cubic easing produces natural movement that starts slow, accelerates, and slows down at the end. Bounce easing adds a playful, elastic effect where the animation overshoots and bounces back, creating an exaggerated cartoon-like quality.

Expressions

The system animates between different facial expressions that can be changed from the simulation. Each expression is defined by a set of blend shape weights that control individual facial features like eyebrows, mouth, and cheeks.
