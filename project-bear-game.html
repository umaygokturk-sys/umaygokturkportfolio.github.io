Bear Game

Three.js WebGL GLSL Shaders JavaScript

A browser-based 3D action game built with Three.js and WebGL. Control an orb to collect energy spheres while evading a laser-armed bear. Features real-time skeletal animation, custom GLSL shaders, and dynamic particle effects.

Gameplay

Navigate the orb across the cobblestone arena to collect green energy spheres. The bear tracks your movement with its eyes and fires lasers when you enter its range. The laser range changes randomly in time, so you never know. If the lasers hit you for too long, your orb overloads and explodes. Collect 5 spheres to defeat the bear.

Controls

  • W / S — Move forward / backward
  • A / D — Move left / right
  • Q / E — Move up / down
  • R — Restart game
  • Mouse — Orbit camera

Graphics Techniques

Eye Tracking with LookAt

The bear's eyes continuously track the player's orb using lookAt() transformations. Each eye socket calculates the direction vector to the target and orients accordingly, creating a convincing gaze-following effect.

Custom GLSL Shaders

The laser beams and eye materials use custom vertex and fragment shaders written in GLSL. The laser shader creates a glowing, semi-transparent beam effect with proper depth blending.

Dynamic Laser Geometry

Lasers are rendered as cylinder geometries that dynamically scale based on the distance between the eye and the target. The laser origin is positioned at the eye socket, and the length is calculated in real-time using world-space distance measurements.

Particle System

Explosions use a custom particle system that spawns spheres with randomized velocity vectors. Each particle has independent lifetime and opacity decay, creating a satisfying explosion effect when the orb overloads or when the bear is defeated.

Color Interpolation

The orb's color shifts from yellow to magenta as it absorbs laser energy, implemented using linear interpolation for smooth RGB interpolation based on the current size ratio.

Technical Implementation

  • Real-time 3D rendering at 60fps using WebGL 2.0
  • Custom ShaderMaterial for laser beam effects
  • Dynamic geometry scaling for distance-based visuals
  • Collision detection using world-space distance calculations
  • Randomized laser range for unpredictable difficulty
  • PBR materials with texture maps (diffuse, normal, roughness, AO)

Core Technologies

  • Three.js — 3D rendering framework
  • WebGL 2.0 — GPU-accelerated graphics
  • GLSL — Custom vertex and fragment shaders
  • JavaScript ES6 — Game logic and animation loop
